## Адаптивность и UX-принципы

### Мобильная версия
- **Главный принцип**: Одноколоночная компоновка
- **Навигация**: Сворачивающееся меню (бургер)
- **Кнопки**: Увеличенные для пальцевого управления
- **Контент**: Вертикальное расположение блоков

### Цветовая схема
- **Основные цвета**: Синий (#0066CC), золотой (#FFD700)
- **Фон**: Темные оттенки для игровой атмосферы
- **Акценты**: Зеленый для успеха, красный для ошибок
- **Текст**: Белый на темном фоне, темный на светлом

### Иконография
- **Система**: Иконки 12x12px для текста, 32x32px для заголовков
- **Стиль**: Pixel-art стиль в духе WoW
- **Категории**: Игровые предметы, UI элементы, статусы

## Технические особенности

### Структура URL
```
/ - Главная страница
/login - Авторизация  
/cabinet - Кабинет игрока
/character/{name} - Страница персонажа
/vote - Голосование
/news - Новости
/admin - Админ-панель
```

### Состояния интерфейса
- **Неавторизованный**: Форма входа в left_block
- **Авторизованный**: Меню профиля и расширенный функционал
- **Администратор**: Дополнительные пункты меню и админ-панель

### Уведомления
- **Глобальные**: Вверху страницы (награды, системные сообщения)
- **Локальные**: Внутри блоков (ошибки форм, успех операций)
- **Интерактивные**: Возможность скрыть/отметить как прочитанное

Этот wireframe описывает полную структуру интерфейса проекта AnotherWoW с учетом всех основных страниц и компонентов.
